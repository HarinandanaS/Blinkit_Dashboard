# 🛒 Blinkit Sales Dashboard

A complete Sales Dashboard built using **Power BI** and **Python** to analyze Blinkit sales data. This project provides insights into revenue, sales, product performance, ratings, and category-wise analysis through interactive visualizations.

---

## 📌 Project Overview

This dashboard helps analyze:
- Total Revenue
- Total Products
- Average Rating
- Total Quantity Sold
- Revenue by Category
- Sales by Category
- Product-wise Performance
